## Part B - Stretch Problem 

The Vector class represents a mathematical vector using a tuple of numbers. It demonstrates operator overloading through dunder methods.

## Key Learnings

- Dunder methods allow custom objects to behave like built-in Python data types.

- `__add__`,`__sub__`, and `__mul__` enable operator overloading for mathematical operations.

- `__repr__` improves readability when printing objects.

- Custom classes help model real-world mathematical structures in object-oriented programming.
