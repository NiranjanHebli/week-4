# Day 20 - PM - Probability Distribution

## Part A - Concept Application

This section covers descriptive statistics, standardization, outlier detection, and hypothesis testing through practical implementation on generated and real datasets.

## Key Learnings

- Normal distribution models naturally occurring continuous data and its histogram should approximate a bell curve.

- Manual calculation of mean, variance, and standard deviation strengthens understanding of statistical formulas.

- Z-score standardization converts data to a common scale, making comparison easier across features.

- Outliers can be detected effectively by checking extreme standardized values.

- Hypothesis testing uses sample evidence to decide whether an assumed population mean is plausible.

## Part B - Stretch Problem 

This section compares raw and standardized distributions and introduces hypothesis testing to compare two groups. It also explains when standardization is needed and why Z-score transformation is important in machine learning.

## Key Learnings

- A normal distribution can have any mean and standard deviation, while a standard normal distribution always has mean 0 and standard deviation 1.

- Standardizing data changes the scale of values but keeps the distribution shape unchanged.

- Standardization is useful when features have different units or ranges.

- Z-score is important in machine learning because it improves feature comparability and helps many algorithms converge more effectively.
