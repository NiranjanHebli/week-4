# Day 22 - AM - EDA Using Pandas 

## Part A - Concept Application

This task demonstrates selecting, filtering, and summarizing data from the Iris Dataset using pandas.

## Key Learnings

- loc selects data using labels

- iloc selects data using index positions

- Filtering extracts rows based on multiple conditions

- describe() summarizes central tendency and spread

- Histogram reveals frequency distribution

- Bar plot compares category wise averages

- Line chart shows trends across observations

- KDE plot provides smooth density estimation
