# Day 19 - AM - Python Fundamentals

## Part A - Concept Application

This task demonstrate how to implement basic statistical operations in Python using both manual logic and functional programming techniques. Mean, median, and variance are computed using loops and then rewritten using higher-order functions. A student marks dataset is analyzed using list comprehension and standard deviation is computed manually and validated using NumPy. Finally, two sample groups are compared statistically.

## Key Learnings

- Learned how mean, median, variance, and standard deviation are calculated from first principles.
- Understood the difference between iterative programming (loops) and functional programming approaches.
- Used map, filter, and reduce to simplify statistical computations.
- Applied list comprehension for efficient grade generation.
- Verified manual statistical results using NumPy for reliability.
- Compared two datasets using descriptive statistics to understand differences in distribution.

## Part B - Stretch Problem

In this section we perform two-sample t-test compares the means of two independent samples and determines whether the observed difference is statistically significant or due to random variation. The function computes the t-statistic by dividing the difference in means by the standard error and compares it with a critical threshold at a significance level of 0.05. If the t-statistic exceeds the critical value, the two groups are considered significantly different.

## Key Learnings

- Learned how hypothesis testing helps compare two groups statistically.

- Mean and variance are essential inputs for computing the t-statistic.

- A larger t-value indicates stronger evidence of difference between samples.

- Significance level (α = 0.05) controls the threshold for decision making.

- SciPy can be used to validate manual t-test results efficiently.
