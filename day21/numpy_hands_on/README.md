# Day 20 - AM - Probability Distribution

## Part A - Concept Application

This task covers creating and manipulating arrays of different dimensions using NumPy. It covers indexing, slicing, vectorized operations, and broadcasting.

## Key Learnings

- NumPy arrays support multi-dimensional data representation, making it easy to work with vectors, matrices, and higher-dimensional datasets.

- Indexing and slicing allow fast extraction of elements, rows, columns, and subarrays without manual iteration.

- Vectorized operations perform arithmetic on entire arrays at once, making code shorter and much faster than loops.

- Statistical functions like mean(), var(), and std() provide quick summary measures of data distribution.

- Broadcasting enables operations between arrays of different shapes by automatically expanding compatible dimensions.

- Array transformations such as squaring, cubing, clipping negatives, and normalization are efficient using built in NumPy operations.

## Part B - Stretch Problem 

This task covers matrix computation using NumPy for efficient linear algebra operations, including matrix multiplication, transpose, and determinant calculation. It also compares execution speed between traditional Python loops and NumPy operations on large arrays.


## Key Learnings:-

- NumPy provides optimized functions for matrix multiplication, making multi-dimensional calculations efficient and concise.

- The transpose operation swaps rows and columns, which is useful in matrix transformations and data rearrangement.

- The determinant helps determine matrix properties such as invertibility and is important in solving linear systems.

- Linear equations can be solved directly using numpy.linalg.solve(), avoiding manual algebraic calculations.

- NumPy’s internal implementation is much faster than Python loops because operations are vectorized and executed in optimized low level code.
