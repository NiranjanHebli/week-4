# Day 21 - PM - Machine Learning Fundamentals

## Part A :- Concept Application

This task demonstrates the difference between regression and classification using synthetic datasets. Regression predicts a continuous numeric value, while classification predicts a category or class label. By training simple models and evaluating them manually, we understand how output type, loss calculation, and interpretation differ between the two machine learning problem types.

## Key Learnings:-

- Regression predicts numeric values, classification predicts labels.

- MSE measures regression prediction error.

- Accuracy measures classification correctness.

- Output type determines model choice.

## Part B :- Stretch Problem 

This exercise demonstrates the bias variance tradeoff, where a simple model may underfit and a complex model may overfit.


## Key Learnings:-

- Low complexity models often underfit because of high bias.

- High complexity models may overfit because of high variance.

- Training error alone is not enough to judge model quality.

- Best performance comes from balancing simplicity and flexibility.
